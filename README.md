# debug_log_rtt
Source code for debug text log over Segger RTT

## Support

1. Print text color

| Text | Color |
|------|-------|
| Normal | White |
| Info | White |
| Warning | Yellow |
| Error | Red

2. Assert

|Format | Details |
|-------|---------|
| ASSERT | without print message. |
| ASSERT_FULL | Print message file and line.
